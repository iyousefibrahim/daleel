# Daleel

**Daleel** is a mobile application built with **React Native** and **Expo** that helps users navigate and complete government services efficiently. The app provides detailed service guides, step-by-step instructions, and requirements for each service. Users can track their progress, earn badges for contributions, and interact with community-shared experiences.  

> ⚠️ **Note:** The project is currently under active development.

## Features

- Browse government services by category.
- View detailed service information and step-by-step instructions.
- Track progress for completed services.
- Community-contributed experiences and badges.
- Supabase integration for backend data management.
- Responsive and intuitive UI using Tamagui.

## Tech Stack

- **Frontend:** React Native, Expo, Tamagui  
- **Navigation:** React Navigation (Native Stack & Bottom Tabs)  
- **Backend:** Supabase (PostgreSQL)  
- **State Management & Data Fetching:** React Query  
- **Forms & Validation:** React Hook Form + Zod
